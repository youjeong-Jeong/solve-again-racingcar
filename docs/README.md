# 구현 기능 목록

## 입력
### 경주 할 자동차 이름 입력 받기
---
#### 정상적인 입력
- 5자 이하의 입력
- 쉼표 기준을 구분된 입력
- 2개 이상의 자동차 입력?

#### 예외 상황
- 공백 입력
- 한 개의 자동차 입력
- 5자 이상의 이름 입력


### 시도할 횟수 입력 받기
---
#### 정상적인 입력
- 1 이상의 숫자

#### 예외 상황
- 공백인 경우
- 문자인 경우

### 결과 출력

- 최종 우승자 안내 문구
